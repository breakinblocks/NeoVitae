# Neo Vitae

[![Discord](https://img.shields.io/badge/Discord-Join%20Us-5865F2?logo=discord&logoColor=white)](https://discord.gg/TRxaXaYy42)

A magic mod centered around the power of blood and sacrifice for NeoForge 1.21.1.

## About

Neo Vitae is a comprehensive rewrite that brings blood magic gameplay to modern Minecraft versions with clean, maintainable code and contemporary modding practices.

### What's New

- **NeoForge for 1.21.1** - Built for the latest Minecraft and modding platform
- **Data Components** - Modern data component system replaces legacy NBT
- **DataMaps** - Configurable mechanics through data-driven systems
- **Clean API** - Well-documented API for addon developers
- **Modern Codecs** - Proper serialization using Minecraft's codec system
- **Active Maintenance** - Regular updates with clear release cadence

### Goals

- Stability first - we prioritize crash fixes and world safety
- Modern, maintainable codebase
- Healthy contribution workflow
